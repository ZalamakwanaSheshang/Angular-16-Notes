Pipes in Angular

A Pipe in Angular is a simple function you can use in component templates to transform, format, or filter data for display without changing the component class itself. They are used to improve user experience by presenting data in a readable format.

Think of a pipe as a simple filter that takes an input value, processes it, and returns a new, transformed value. The syntax uses the vertical bar (|).

Types of Pipes

Angular primarily has two categories of pipes: Built-in Pipes and Custom Pipes. Within the Built-in category, pipes can be classified as Pure or Impure.

Type Description
Built-in Pipes Pipes provided by Angular framework (e.g., DatePipe, CurrencyPipe).
Custom Pipes Pipes created by the developer to handle specific application logic.

1. Built-in Pipes (Examples)

These pipes are part of the Angular framework and cover common data formatting needs.

DatePipe

The DatePipe formats a date according to locale rules and specific formats.

// Component (.ts)
today = new Date(); // e.g., Thu Oct 16 2025 22:29:14 GMT+0530
<!-- Html Code -->
<p>Default: { { today | date } } </p>
<p>Full Date: { { today | date:'fullDate' } } </p>
