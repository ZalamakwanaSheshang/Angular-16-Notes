Getting Started: Angular Installation and Project Creation

To begin your journey, you need to set up your development environment. Follow these simple steps:

  1. Step 1: Install Node.js and npm

    Angular relies on **Node.js** and its package manager, **npm**.

    • Go to the official Node.js website and download the latest **LTS (Long-Term Support)** version.
    • Run the installer and follow the default prompts.
    • Verify the installation in your terminal/command prompt:
      node -v
      npm -v
      (This should display the installed versions.)

  2. Step 2: Install the Angular CLI

    The Angular Command Line Interface (**CLI**) is a powerful tool for creating, managing, and building Angular applications.

    • Open your terminal or command prompt.
    • Install the CLI globally using npm:
      npm install -g @angular/cli
    • Verify the installation:
      ng version

  3. Step 3: Create a New Angular Project

    The Angular CLI makes generating a new project workspace effortless.

    • Navigate to the directory where you want to create your project:
      cd [Your desired directory]
    • Run the ng new command, replacing my-angular-app with your chosen project name:
      ng new my-angular-app
    • The CLI will prompt you to answer a few configuration questions (e.g., "Would you like to add Angular routing?" and "Which stylesheet format would you like to use?"). Select your preferred options.
    • The CLI will create the project folder and install all necessary dependencies.

  4. Step 4: Run Your Application

    • Navigate into your new project directory:
      cd my-angular-app
    • Start the development server:
      ng serve --open
    • The --open (or -o) flag automatically opens your new application in your browser, typically at http://localhost:4200/. You are now ready to start coding!
